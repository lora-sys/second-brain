## How to use this index

1. **Starting a new feature**: read `roadmap.md` for context, then read the linked PRD/MVP/feature spec.
2. **Reviewing a PR**: start at the linked Issue, follow the Evidence path, read the change-summary.
3. **Understanding a decision**: ADRs are in `decisions/`. Cross-cutting timeline in `memory/decisions.md`.
4. **Recovering from a long pause**: read `PROJECT_STATUS.md` → last session summary → ADRs.

The personal cognitive OS for a single user who wants their notes, tasks, and reflection to actually work together.

Second Brain OS is a Tauri desktop shell + Obsidian-vault engine. Your data lives in plain Markdown files you control. The app surfaces the right entity at the right time, captures new ones in seconds, and quietly learns from your behavior. No accounts. No cloud. No telemetry. Just you and your second brain.

Features

An Obsidian vault, but with a brain.

Every entity is a plain Markdown file. The app is a layer on top that understands the relationships.

4 entity types, one keyboard

人物 · 任务 · 项目 · 链接. Each is a plain .md file. The keyboard shortcut ⌘N creates new ones; smart mentions cross-link them.

Cockpit view, not a file tree

Open the app, see today's reflection · achievements · focus. The right rail shows tasks and what's coming. Bottom row is memory + bookmarks. You don't navigate files — the app surfaces what's relevant.

Full-text search across the vault

Rust-backed search that ranks title matches above body matches, exact above substring. Type in the search bar; the right entity comes up.

Atomic writes + file locks

Every change is atomic (tmp + rename) and concurrency-safe (per-directory lock). Even if you close the laptop mid-save, your data is consistent.

9 Tauri commands, full local API

config_get · config_set · vault_list · vault_read · vault_create · vault_update · vault_delete · vault_search · vault_list_by_type. Every operation in the app maps to one.

Light + dark, system-aware

Three themes (浅色 · 深色 · 复古) wired to your OS preference. CSS custom properties mean a single set of tokens, no duplication.

What it is, in one diagram.

Install

Three commands. Linux first.

The .deb package targets Debian / Ubuntu. The Tauri build pipeline produces AppImage on ubuntu-22.04 (GitHub Actions). Mac / Windows are filed for v0.5+.

01 / DOWNLOAD

Get the .deb

Grab the latest release from GitHub Releases. Currently 0.4.0-alpha. The .deb is the build artifact for Debian / Ubuntu / Mint.

02 / INSTALL

dpkg -i

sudo dpkg -i second-brain_0.4.0-alpha_amd64.deb. The desktop entry appears in your app launcher. Vault is configured on first launch.

03 / USE

Open Cockpit

Open the app, click the 日程 nav, see today's panel. Create a task with due: 2026-07-14 and it appears in tomorrow's bucket.
